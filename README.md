# PhonixKit

Минимальный Phoenix компонент для создания красивых welcome страниц.

## Установка

Добавьте `phonix_kit` в список зависимостей в `mix.exs`:

```elixir
def deps do
  [
    {:phonix_kit, "~> 0.1.0"}
  ]
end
```

## Использование

Просто используйте компонент в любом Phoenix шаблоне:

```elixir
<PhonixKit.welcome title="Добро пожаловать!" />
```

### Параметры

- `title` (обязательный) - заголовок страницы
- `subtitle` (опциональный) - подзаголовок, по умолчанию "Phonix Kit успешно установлен"
- `class` (опциональный) - дополнительные CSS классы

### Примеры

```elixir
# Базовое использование
<PhonixKit.welcome title="Hello World" />

# С кастомным подзаголовком
<PhonixKit.welcome title="Мой проект" subtitle="Готов к работе!" />

# С дополнительными классами
<PhonixKit.welcome title="Welcome" class="custom-height" />
```

## Внешний вид

Компонент создает полноэкранную страницу с:
- Градиентным фоном (синий → фиолетовый)
- Центрированным текстом
- Адаптивным дизайном
- Tailwind CSS стилями

## Требования

- Phoenix LiveView ~> 0.18
- Tailwind CSS (для стилей)
